# UCalendar 日历

:::warning{title=写在前面}
`UCalendar` 是基于 antd 的 `Calendar` 组件进行开发的拓展组件，支持 `Calendar` 的所有特性（除`cellRender`），同时还支持农历、节假日的展示；
:::
